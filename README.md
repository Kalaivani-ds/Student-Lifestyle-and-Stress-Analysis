# Student Lifestyle and Stress Analysis

## Overview

This project analyzes a student lifestyle and stress dataset using Python.  
The main goal is to understand how different lifestyle factors such as sleep hours, study hours, and GPA affect student stress levels.

The project uses:
- Python
- Pandas
- Matplotlib

---

## Features

- CSV data analysis
- Missing value checking
- Statistical summary
- Stress level visualization
- Sleep vs stress analysis
- Study hours vs GPA analysis
- Correlation heatmap

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- VS Code
